Brief Description

Released on September 20th, 2019. The "Box Puzzle" chamber was added, later renamed to "Block Puzzle" in 1.4.3, a new type of prop called the "No-Gravity" props were added, and a floating animation was added to the Teleporter cubes.

Patch Notes

-Added a Box Puzzle level in the hub -Fixed the physics test, now it doesn't go at 2 fps -Added a new type of prop which for now it's only seen in the Box Puzzle, no-gravity prop. -Fixed some minor bugs -Added a floating animation to the teleportation cubes
