1.7

Brief Description

Released on June 19th, 2020. After a six-month-long pause, development was resumed. A new chamber was added, The Café NPCs were finally added after being planned for a long time. The ability to push and pull props was implemented. The Main Menu was added, despite not being listed in the patch notes. The camera Field Of View was increased. New levels for the block puzzle were added. Upon entering a chamber, a title card is now shown.

Patch Notes

-You now have the ability to push and pull props from a distance! While looking at props,    °Press Q to attract the prop to you.    °Press F to impulse the prop away. -Removed some irrelevant stuff -NEW LEVELS FOR THE BLOCK PUZZLE AND NEW MECHANICS! -Increased FOV to 75 (previously 60) -Fixed a thing that made you able to see through walls if you stood near them -Testing some stuff such as music card system in the Main Chamber    °Enter the main chamber, go outside the room you spawn in and go to the one on the left. -Added Title Cards when entering a Chamber -Added a Café chamber -Added a dialogue/textbox system to use for NPCs -Added an [E] text below the crosshair when an item is interactable -Added small changes and fixes to some chambers -Added the Main Menu (Not listed in the original patch notes)
